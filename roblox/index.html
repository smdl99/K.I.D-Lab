Roblox

In this course, students learn how to create Roblox games from scratch using Roblox Studio. They build 3D worlds, design gameplay mechanics, and use Lua scripting to make their games interactive.

Age

8–12 years

Level

Beginner

Duration

32 lessons

start today
Game Worlds

Build your own Roblox maps and environments using Roblox Studio tools, terrain, and 3D objects.

Obby Design

Create obstacle courses with moving platforms, traps, parkour jumps, and challenging levels.

Lua Coding

Write scripts that control doors, platforms, and how players interact with your game environment.

Game Mechanics

Add jump pads, checkpoints, puzzles, and interactive gameplay elements and mechanics.

NPCs & Vehicles

Create characters that interact with players and build vehicles such as racing cars and tracks.

Robux Systems

Learn how Roblox games use Game Passes, rewards, and other systems to earn Robux inside games.
